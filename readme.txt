ダウンロードしたソースコードは解凍すると各サンプルごとに分かれています。
それぞれのサンプルのフォルダ内の [example.js] ファイルをお使いのコンピュータでのテキストエディタを使って、編集、変更、保存し、同じフォルダ内の [index.html] ファイルをウェブブラウザで見ることで、その変更内容を確認します。

テキストエディタは一般的なものでは、Macだと mi( https://www.mimikaki.net/ ) や atom( https://atom.io/ ) など、Windowsだと Visual Studio Code( https://azure.microsoft.com/ja-jp/products/visual-studio-code/ ) や atom( https://atom.io/ ) などがあります。
